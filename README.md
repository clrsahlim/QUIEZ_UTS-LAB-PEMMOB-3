Nama Aplikasi: QUIEZ
Deskripsi Aplikasi: QUIEZ merupakan aplikasi quiz interaktif simple yang menarik dan mudah digunakan, tersedia dalam beberapa kategori menarik untuk menguji pengetahuan umum pengguna. Kategori pertanyaan dibagi menjadi empat yaitu, pengetahuan umum, sejarah yunani, tebak lagu, dan tebak penyanyi. Aplikasi terdiri dari 5 halaman utama, yaitu splash screen, category screen (halaman awal untuk input username dan memilih kategori), quiz screen (halaman pertanyaan sesuai kategori), score screen (halaman untuk menampilkan jumlah score pengguna), dan result detail screen (halaman untuk menampilkan pertanyaan yang benar dan salah).
